# Hankki : The Dining Hosting Platform

## Deployment
### Installation
* [Docker](https://www.docker.com/community-edition)
* [Docker Compose](https://docs.docker.com/compose/)

### Build
``` shell
$ docker-compose build
```

### Run
* -d: Detached mode

``` shell
$ docker-compose up -d
```

### Clean
``` shell
$ docker-compose down
```

### GUI Tool for Docker
* [Kitematic](https://kitematic.com/)
