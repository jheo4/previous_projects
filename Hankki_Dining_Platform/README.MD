# Team 4men Github

## Project name : 한끼줍쇼

This repository is for 2018 capstone design project

## Deployment
### Installation
* [Docker](https://www.docker.com/community-edition)
* [Docker Compose](https://docs.docker.com/compose/)

### Build
``` shell
$ docker-compose build
```

### Run
* -d: Detached mode

``` shell
$ docker-compose up -d
```

### Clean
``` shell
$ docker-compose down
```

### GUI Tool for Docker
* [Kitematic](https://kitematic.com/)
